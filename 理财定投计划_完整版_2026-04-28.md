## 九、心态指南

1. **不看短期涨跌**：每天看净值是焦虑的来源。每周看一次就够了。
2. **定投是长跑**：31年的计划，前5年的波动不重要。重要的是坚持。
3. **考编优先**：投资很重要，但考编回重庆、稳定收入是更大的杠杆。
4. **不要和别人比**：每个人的起点不同。你的计划是为你自己设计的。
5. **允许犯错**：如果某个月忘记定投，下个月补上就好。完美主义是敌人。
6. **享受过程**：看着资产一点点增长，本身就是一种快乐。

## 附录：关键公式

**复利公式：**
```
未来价值 = 本金 × (1 + 年化收益率)^年数 + 定投 × [(1 + r)^n - 1] / r
```

**实际购买力（扣除通胀）：**
```
实际价值 = 名义价值 / (1 + 通胀率)^年数
```

**安全提取率（退休后可用）：**
```
每年可提取 = 总资产 × 4%
（4%法则：历史数据显示，按此比例提取，30年内不会耗尽资产）
```
