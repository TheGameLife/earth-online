#### 2.1 睡眠——一切的底盘

你不需要"早睡早起"的口号。你需要的是一个**能做到的系统**。

**当前问题：**
- 工作日：8点起，23-24点睡 → 睡眠7-8小时，但质量不好
- 周末：10点起，1点睡 → 生物钟混乱
- 你自己说：22:30睡最舒服

**目标作息：**

| | 工作日 | 周末 |
|---|---|---|
| 起床 | 7:30 | 8:30 |
| 睡觉 | 22:30 | 23:00 |
| 睡眠时长 | 9小时 | 8.5小时 |

**如何实现（原子习惯法）：**

不是"我要早睡"——这是意志力，会失败。

而是设计一个**无需意志力的晚间系统**：

```
21:30 → 洗漱完毕（固定流程，不需要想）
21:40 → 放下手机，放到够不到的地方（环境设计）
21:40-22:00 → 自由时间：看书/写字/冥想/拉伸（低刺激活动）
22:00 → 上床，关灯
22:00-22:30 → 如果睡不着，闭眼听轻音乐/白噪音
22:30 → 入睡
```

**关键设计：**
1. **手机隔离法**：21:40后手机放客厅/书桌上，不带上床。买一个实体闹钟。
2. **两分钟规则**：不想早睡时，告诉自己"就洗漱一下"。洗漱完自然就会进入流程。
3. **身份认同**：不说"我要早睡"，说"我是一个重视睡眠的人"。
4. **周末不崩坏**：周末起床不超过9:00，保持生物钟稳定。

**追踪方式：**
- 在滴答清单设置每日打卡：✅ 22:30前入睡
- 连续7天→奖励自己一个小东西（一本书、一束花）
- 断了→**Never Miss Twice**（原子习惯核心原则：允许断一次，绝不断第二次）

4. **用"做什么"替代"不做什么"**：
   - 不想刷手机时 → 翻开书（床头永远有一本开着的书）
   - 无聊时 → 写10分钟字
   - 累了时 → 闭眼拉伸5分钟

**允许自己不完美。** 如果某天刷了2小时，不要自责。第二天回到正轨就好。

## 第四章：原子习惯的核心原则

### 1. 让好习惯显而易见
- "我要运动" → 改成 "每天早上起床后，我会在客厅做八段锦"
- 把瑜伽垫铺在客厅中央，让你不得不看到它

### 2. 让好习惯有吸引力
- 运动后奖励自己一杯好喝的（无糖的）
- 学习后奖励自己10分钟喜欢的视频

### 3. 让好习惯简单易行
- "每天运动30分钟" → "每天八段锦8分钟"
- "每天学习2小时" → "每天做10道题"
- **两分钟规则：任何新习惯，前两分钟版本都要简单到不可能失败**

### 4. 让好习惯令人满足
- 在滴答清单打卡，看到连续记录的感觉很好
- 每完成一周，给自己一个小奖励

### 5. 身份认同
- 不说"我要早睡" → 说"我是一个重视睡眠的人"
- 不说"我要运动" → 说"我是一个每天动一动的人"
- 不说"我要考编" → 说"我是一个正在准备事业编考试的人"
- **每一次行动，都是在给你想成为的那个人投票**

### 6. Never Miss Twice
- 断了一天？没关系。
- 断了两天？不允许。
- 允许不完美，但不允许连续放弃。

## 第五章：高效能人士的七个习惯（你的版本）

**你的版本：** 不再等"有状态了再做"。状态是做出来的，不是等出来的。明天早上，不管想不想，先做8分钟八段锦。

### 习惯2：以终为始
**你的版本：** 35岁的画面。那个阳光房间里的你。每次想放弃的时候，想一秒钟她。她就是你。

### 习惯3：要事第一
**你的版本：** 考编学习 > 刷手机 > 无意义的社交。每天问自己：今天最重要的3件事是什么？做完了，今天就没白过。

### 习惯4：双赢思维
**你的版本：** 和男友的关系。各自有空间，在一起时舒服。不讨好，不冷漠。刚刚好。

### 习惯5：知彼解己
**你的版本：** 先理解自己。写字、独处、读书。理解了自己，才能和别人好好相处。

### 习惯6：统合综效
**你的版本：** 运动+学习+读书。这三件事互相促进。运动后脑子清醒，学习效率更高。读了好书，心态更好，人际关系更舒服。

### 习惯7：不断更新
**你的版本：** 身体、精神、社交、理财——四个维度，每周检查一次。哪个维度偏弱，下周多关注一点。

## 第七章：当你想放弃的时候

你会想放弃的。这很正常。

当你想放弃的时候，做这三件事：

1. **停下来，深呼吸3次。** 不要做任何决定。
2. **想一想：** 我是累了，还是真的不想做了？如果是累了，休息，但不放弃。
3. **把目标缩小到最小版本：** 今天不想学30分钟？学5分钟就好。不想运动？做1分钟拉伸就好。**做了5分钟，你就赢了。**

记住高二的你。那个可以每天严格执行计划的她，还在你身体里。她只是被疲惫和恐惧盖住了。

**她不需要你"变得更强"。她只需要你开始做一件小事。**

明天早上。8分钟八段锦。
